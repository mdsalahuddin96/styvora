# 👗 Styvora – Modern Fashion Store

<div>

**A modern and elegant fashion e-commerce storefront built with React and Tailwind CSS.**

Designed with a luxury shopping experience in mind, Styvora allows users to browse products, search collections, filter by category, view product details, and manage a shopping cart using local data.

</div>

---

## 🌐 Live Demo

🔗 **Live Site:** [https://styvora.vercel.app](https://styvora.vercel.app/)

# ✨ Features

-  Elegant and modern fashion storefront
-  Search products by name
-  Filter products by category
-  Detailed product page

# 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/mdsalahuddin96/styvora.git
```

---

### 2. Navigate to the project

```bash
cd styvora
```

---

### 3. Install dependencies

Using npm

```bash
npm install
```

Or using yarn

```bash
yarn
```

Or using pnpm

```bash
pnpm install
```

---

### 4. Start the development server

```bash
npm run dev
```

---

### 5. Open your browser

http://localhost:3000
```

---

# 📦 Build for Production

```bash
npm run build
```
